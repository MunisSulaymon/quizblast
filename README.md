# QuizBlast 🎮
A real-time multiplayer quiz game inspired by Kahoot!

## Live Demo
https://quizblast.onrender.com

## Features
- Real-time multiplayer via Socket.io
- 6-digit PIN system
- Speed-based scoring with streak bonuses
- Live leaderboard
- Mobile friendly
- Confetti celebration

## Tech Stack
- Node.js + Express
- Socket.io
- Vanilla HTML/CSS/JavaScript

## Run Locally
npm install
npm start
Open http://localhost:3000

## Deploy on Render
1. Push to GitHub
2. Connect repo on render.com
3. Set build command: npm install
4. Set start command: npm start
5. Deploy!
